# Differentially-Private-Federated-DRO
This repository contains the code used in the paper **"Differential Privacy for Distributionally Robust Multiclass Classification"**.
The code implements and evaluates different federated multiclass classification methods, including:

- DP-Fed DRO: the proposed differentially private federated distributionally robust optimization method
- DP-Fed ERM: a differentially private federated empirical risk minimization benchmark
- Non-Private Fed DRO: a non-private federated DRO benchmark

The experiments include privacy-utility comparisons, convergence analysis, and scalability studies with respect to the number of clients and the number of classes.
